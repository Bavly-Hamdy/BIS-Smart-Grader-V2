## ï¿½ About The Project

**BIS Smart Grader** is an enterprise-grade academic management system engineered to transform how university faculties manage assessments. By bridging the gap between traditional grading and modern AI capabilities, we provide a secure, efficient, and visually immersive environment for educators.

We have moved beyond functional utility to create an **Experience**. Every interaction, from the gradient-infused login screen to the interactive data visualization dashboards, is crafted to be intuitive, accessible, and professional.

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/bis-smart-grader.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Set up your `.env` file with Firebase credentials
    ```js
    VITE_FIREBASE_API_KEY=...
    VITE_FIREBASE_AUTH_DOMAIN=...
    // ... other firebase config
    ```
4.  Run the application
    ```sh
    npm run dev
    ```
